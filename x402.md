# AgentCash Integration for OpenClaw

Wire up agentcash so your agent can make paid API calls (USDC on Base) — data enrichment, web research, image/video generation, social scraping, and more.

## Prerequisites

- OpenClaw gateway running (complete [EC2.md](EC2.md) first)
- SSH access to your EC2 instance

## Overview

OpenClaw doesn't natively support MCP servers yet. We bridge this using:

1. **mcporter** - CLI tool to call MCP servers
2. **agentcash** - MCP server for x402 payment protocol
3. **AgentCash Skills** - Teach the agent how to use agentcash tools

## 1. Install mcporter

SSH into your EC2 instance:

```bash
ssh ubuntu@<your-ec2-ip>
```

Install mcporter globally:

```bash
sudo npm install -g mcporter
```

## 2. Configure agentcash MCP

Add the agentcash MCP server to mcporter:

```bash
mcporter config add agentcash \
  --command "npx" \
  --arg "-y" \
  --arg "agentcash@latest" \
  --scope home
```

Verify it works:

```bash
mcporter call agentcash get_wallet_info
```

This creates a wallet at `~/.agentcash/wallet.json` and shows your address + balance.

## 3. Install AgentCash Skills

Skills teach the agent how to use mcporter for agentcash operations. The `skills` CLI is fetched automatically via npx:

```bash
npx skills add Merit-Systems/agentcash-skills/mcp --all --yes
```

This downloads skills to `~/.skills/`.

Skills installed:
- **agentcash-wallet** - Balance, deposits, invite codes, discovering & calling any x402 API
- **data-enrichment** - Person, company, & influencer profiles; email verification (Apollo, Clado, Hunter)
- **web-research** - Web search & scraping (Exa, Firecrawl)
- **local-search** - Places & business info (Google Maps)
- **social-intelligence** - X/Twitter & Reddit search (Grok, Reddit)
- **social-scraping** - Scrape profiles, posts, followers across 6 platforms (StableSocial)
- **media-generation** - AI image & video generation (StableStudio)
- **upload-and-share** - File hosting & public URLs (StableUpload)
- **email** - Send emails, forwarding inboxes (StableEmail)
- **phone-calls** - AI phone calls (StablePhone)
- **news-shopping** - News & product search (Serper)
- **people-property** - People & property lookup (Whitepages)

## 4. Link Skills to Workspace

Create a symlink so OpenClaw finds the skills:

```bash
mkdir -p ~/.openclaw/workspace
ln -sf ~/.skills ~/.openclaw/workspace/skills
```

## 5. Restart Gateway

```bash
sudo systemctl restart openclaw
```

## 6. Verify

Check the gateway loaded the skills:

```bash
sudo journalctl -u openclaw -n 50 --no-pager | grep -i skill
```

## 7. Fund Your Wallet

Your agentcash wallet needs USDC on Base to make paid API calls.

Get your wallet address:
```bash
mcporter call agentcash get_wallet_info
```

Fund options:
- **Redeem an invite code:** `mcporter call agentcash redeem_invite --code YOUR_CODE`
- **Deposit USDC:** Send USDC on Base to the displayed wallet address. Even $5-10 is enough for testing.

## Usage

Once funded, your agent can:

```
"Search for information about Sam Ragsdale"
"Generate an image of a robot"
"Scrape the content from https://example.com"
"Find CEO contacts in San Francisco"
"Search Twitter for AI agent news"
```

The agent will use mcporter to call the agentcash MCP, which handles USDC payments automatically.

## Troubleshooting

### mcporter not found
```bash
which mcporter
sudo npm install -g mcporter
```

### Wallet not created
```bash
mcporter call agentcash get_wallet_info
ls -la ~/.agentcash/wallet.json
```

### Skills not loading
```bash
# Check symlink
ls -la ~/.openclaw/workspace/skills

# Recreate if broken
rm ~/.openclaw/workspace/skills
ln -sf ~/.skills ~/.openclaw/workspace/skills

# Restart gateway
sudo systemctl restart openclaw
```

### Insufficient balance
```bash
mcporter call agentcash get_wallet_info
# Fund with USDC on Base network or redeem an invite code
```

## Security Notes

- Your agentcash wallet private key is stored at `~/.agentcash/wallet.json`
- Back up this file if you fund it with significant USDC
- The wallet is only accessible to the ubuntu user
- Consider using a dedicated wallet with limited funds
